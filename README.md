# Neo Missio - Sistema de Gestão Social

Sistema completo de gestão para o projeto social Neo Missio, focado no gerenciamento de atividades, alunos, professores e coordenação.

## Funcionalidades principais

- Dashboard administrativo e para responsáveis.
- Gestão de matrículas e atividades.
- Relatórios financeiros e acadêmicos.
- PWA (Progressive Web App) para acesso mobile facilitado.

## Desenvolvimento

Para rodar o projeto localmente:

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse em `http://localhost:8080`.

## Deploy

O projeto está configurado para deploy automático na Vercel vinculado ao repositório GitHub.
Domínio oficial: [sistema.neomissio.com.br](https://sistema.neomissio.com.br)
